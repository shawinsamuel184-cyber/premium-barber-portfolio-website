## [1.0.0] — 2025-01-xx

### Added
- Initial public release of **C-CUTZ** premium barber portfolio.
- Cinematic dark warm-copper theme (`#0A090C` background, `#D4A373` copper accent, cream text).
- Lenis smooth scrolling.
- GSAP + ScrollTrigger animated zig-zag **STORYTIME** timeline.
- Framer Motion animations for hero, cards, testimonials, accordion, marquee.
- Preloader with animated progress bar and logo.
- Custom copper cursor (desktop only).
- Top scroll-progress indicator.
- **Hero** section with portrait, headline "HI, I'M LAURIN", tagline and Book Now CTA.
- **58 Cuts** infinite dual-direction marquee portfolio with numbered cards and automatic image fallback.
- **Get an Idea of My Work** collage section with 3-step process (Consultation → The Cut → Final Style) and copper arrow connectors.
- **Services** grid with 6 premium service cards (price + duration).
- **Testimonials** carousel with featured center quote and side previews, auto-rotating every 5 seconds.
- **Book an appointment** section with calendar, time-slot selector, service select, and form UI.
- **FAQ** accordion with smooth expand/collapse.
- **Footer** with contact info, hours, and social links.
- Mobile / tablet / desktop responsive layouts.
- Rounded-corner UI language (rounded pills, rounded-3xl cards).
- SVG image fallback shown automatically if any image fails to load.
- Documentation:
  - `README.md`
  - `docs/INSTALLATION.md`
  - `docs/DEPLOYMENT.md`
  - `docs/CUSTOMIZATION.md`
  - `docs/CONTRIBUTING.md`
  - `docs/CHANGELOG.md`
