# Image Replacement Guide

## Folder Structure
```
public/images/
├── 58-cuts/          # 58 haircut style images
├── process/          # Haircut process images
├── services/         # Service images
├── hero.*            # Hero section background (any extension)
└── portrait.*        # Laurin's portrait photo (any extension)
```

## Supported Image Formats
You can use **any common image format**:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`
- `.gif`
- etc.

The code will automatically try different extensions if the first one fails!

## Image Files to Replace

### 1. 58 Cuts (`public/images/58-cuts/`)
Add 58 images named (with any extension):
- `01`, `02`, ..., `58`
  (Example: `01.jpg`, `02.png`, `03.webp`, etc.)

### 2. Process (`public/images/process/`)
Add 3 images named (with any extension):
- `step1` (Consultation)
- `step2` (The Cut)
- `step3` (Final Style)

### 3. Services (`public/images/services/`)
Add 6 images named (with any extension):
- `signature-cut`
- `cut-beard`
- `hair-wash-style`
- `premium-grooming`
- `kids-cut`
- `luxury-package`

### 4. Hero & Portrait
Add to `public/images/`:
- `hero` (hero background, any extension)
- `portrait` (Laurin's photo, any extension)
